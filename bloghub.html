BlogHub

A full-stack blog platform with Laravel backend, role-based access control, and MySQL database for content management.

Overview

Built a complete blogging application using Laravel framework with features including user authentication, article creation/editing, category management, and admin dashboard. Demonstrates proficiency in PHP, Laravel architecture, RESTful API design, and MySQL database design.

Key Features

  • Authentication & Authorization: User registration, login, and role-based permissions (Author, Admin)
  • Article Management: CRUD operations with Rich text editor, categories, and tags
  • Database Design: MySQL schema with users, articles, categories, and relationships
  • Admin Dashboard: Analytics, user management, and moderation tools
  • API Endpoints: REST API for article retrieval, filtering, and search
  • SEO Optimization: Slug generation, meta tags, and sitemap generation

Tech Stack

Backend: Laravel 10, PHP 8.1 | Database: MySQL | Frontend: Blade templates, CSS3 | Tools: Composer, Artisan, Laravel Tinker

Laravel Route & Controller (snippet)

// routes/web.php
Route::middleware('auth')->group(function () {
    Route::resource('articles', ArticleController::class);
    Route::post('/articles/{article}/publish', [ArticleController::class, 'publish']);
});

// app/Http/Controllers/ArticleController.php
public function store(StoreArticleRequest $request)
{
    $article = auth()->user()->articles()->create($request->validated());
    return redirect()->route('articles.show', $article);
}

Database Migration (snippet)

// database/migrations/create_articles_table.php
Schema::create('articles', function (Blueprint $table) {
    $table->id();
    $table->foreignId('user_id')->constrained()->onDelete('cascade');
    $table->string('title');
    $table->string('slug')->unique();
    $table->longText('content');
    $table->string('featured_image')->nullable();
    $table->enum('status', ['draft', 'published'])->default('draft');
    $table->timestamps();
});

Eloquent Model with Relations (snippet)

// app/Models/Article.php
class Article extends Model
{
    protected $fillable = ['title', 'slug', 'content', 'status'];
    
    public function author()
    {
        return $this->belongsTo(User::class, 'user_id');
    }
    
    public function categories()
    {
        return $this->belongsToMany(Category::class);
    }
    
    public function scopePublished($query)
    {
        return $query->where('status', 'published');
    }
}
